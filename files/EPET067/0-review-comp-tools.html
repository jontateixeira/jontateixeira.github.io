--- ## Geração de números aleatórios <small> **Gerador de números aleatórios** é a geração de uma sequência de *números* ou *símbolos* que não podem ser razoavelmente previstos melhor do que por uma chance aleatória, geralmente por um gerador de número aleatório de hardware (RNG). Usando <span style="color:blue;">Python</span>, é preciso importar a biblioteca `numpy` e utilizar o módulo random (`np.random`). Enquanto que, para a linguagem <span style="color:purple;">Julia</span> é preciso instalar e usar a biblioteca [Distriburions.jl](https://juliastats.org/Distributions.jl/stable/). </small> --- ### Gerando números aleatórios (<span style="color:blue;">Python</span>) <small> Gerando numeros aleatórios $[0,1]$: * Uniformemente distribuidos ```python import numpy as np x = np.random.rand(10**4); plt.hist(x); ``` * Normalmemente distribuidos ```python import numpy as np x = np.random.randn(10**4); plt.hist(x); ``` </small> --- ### Gerando números aleatórios (<span style="color:purple;">Julia</span>) <small> Gerando numeros aleatórios $[0,1]$: * Uniformemente distribuidos ```julia using Random, Distributions x = rand(10^4); histogram(x) ``` * Normalmemente distribuidos ```julia x = rand(Normal(), 10^4); histogram(x); ``` </small> --- ### Gerando números aleatórios em um intervalo e mesmos números <small> Para gerar aleatoriamente números dentro de um intervalo $[a,b]$ basta fazer: * <span style="color:blue;">Python</span> ```python r = b + (b - a) * np.random.rand(N) ``` * <span style="color:purple;">Julia</span> ```julia r = b + (b - a) * rand(N) ``` `N` deve ser definido (quantidade de números aleatórios a ser gerado) </small> --- ### Gerando os mesmos números <small> Para gerar o mesmo conjunto de números aleatoriamente basta fazer (antes de chamar o gerador): * <span style="color:blue;">Python</span> ```python np.random.seed(N) ``` * <span style="color:purple;">Julia</span> ```julia Random.seed!(N) ``` Onde `N` é a semente do gerador (geralmente será sua matrícula). </small>

--- ### Zero de funções não-lineares código (<span style="color:blue;">Python</span>) <small> Encontraremos o zero da função abaixo: $$f(x) = x^2 - 2x + 8$$ </small> ```python from scipy.optimize import fsolve, root # definindo a função def fun(x): return x**2 - 2* x - 8 ans_root = root(fun, 0.5) ans_fsolve = fsolve(fun, 0.5) print(f" solução pelo root: {ans_root.x[0]}\n solução pelo fsolve: {ans_fsolve[0]}") ``` --- ### Zero de funções não-lineares (<span style="color:purple;">Julia</span>) Há varios métodos de encontrar o zero (raiz) usando o `NLsolve`. Dentre os métodos que podemos listar são: * Secante * Bissecção * Newton Estes métodos, como muitos outros, estão implementados na função `NLsolve` --- ### Zero de funções não-lineares código (<span style="color:purple;">Julia</span>) <small> Encontraremos o zero da função abaixo: $$f(x) = x^2 - 2x + 8$$ </small> ```julia using NLsolve # definindo a função function fun!(a, x) a[1] = x[1] ^ 2 - 2 * x[1] - 8 end ans_solve = nlsolve(fun!, [0.5]) println("Solução: ", ans_solve.zero) ```
